CONTEUDOS:



HTML, sigla para HyperText Markup Language, é a linguagem de marcação fundamental para estruturar o conteúdo de páginas web, sendo o "bloco de construção" da internet. Ela define a estrutura e o significado de textos, imagens, vídeos e outros elementos através de tags, que são interpretadas por navegadores como Chrome ou Firefox para apresentar a página ao utilizado

CSS, ou Folhas de Estilo em Cascata, é uma linguagem usada para descrever a apresentação de documentos HTML ou XML. Em termos simples, o CSS define como os elementos de uma página web devem ser exibidos, incluindo cores, fontes, layout e outros aspectos visuais. O CSS trabalha em conjunto com o HTML, que define a estrutura do conteúdo, enquanto o CSS adiciona estilo e apresentação.

O que é a propriedade Display em CSS


A display: flex; propriedade CSS inicia o módulo Layout de Caixa Flexível, comumente conhecido como Flexbox, para um determinado elemento HTML. Quando aplicada a um elemento contêiner, ela transforma esse elemento em um contêiner flexível, e seus filhos diretos se tornam itens flexíveis. Essa propriedade fornece uma maneira poderosa e eficiente de organizar, alinhar e distribuir espaço entre itens dentro de um contêiner, mesmo quando seu tamanho é desconhecido ou dinâmico.

Display Caracteristicas
block A display: block; declaração CSS renderiza um elemento HTML como um elemento de nível de bloco. Isso significa que o elemento irá: Comece em uma nova linha: Ele ocupará sua própria linha no fluxo do documento, empurrando os elementos subsequentes abaixo dela.
inline A propriedade CSS display: inline; faz com que um elemento se comporte como um elemento de nível embutido. Isso significa que o elemento irá: Não começar em uma nova linha: ele fluirá junto com outro conteúdo embutido dentro do seu elemento pai.
inline-block A propriedade CSS display: inline-blockcombina características dos elementos inlinee block. Principais características de display: inline-block: Fluxos em linha: Semelhante aos inlineelementos, inline-blockos elementos aparecem na mesma linha que o conteúdo ao redor e não forçam uma quebra de linha antes ou depois deles.
